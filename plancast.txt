Цель: реализовать музыкальный чат бот с возможность прослушивания личной музыки, хранением плейлистов под настроения и поиском треков по названию и исполнителю.
План: 
      1. Необходимо создать некоторое хранилище для пользователя, где он сможет сохранять свои песни.
      2. Далее необходимо создать класс "Track", у которого есть поля: название, исполнитель, ссылка на трек.
      3. Затем нужно создать класс "Playlist", в котором будут хранится треки, которые добавит в него пользователь (по умолчанию у пользовтеля будет пустой Плейлист, в который будут добавляться все песни, которые он добавляет)
      4. У пользователя необходимо добавить команды: добавить, удалить, добавить в плейлист песню, посмотреть все плейллисты с песнями
